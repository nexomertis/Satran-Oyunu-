# Fantastik Satranç 3D

Elfler ve Orklar temalı 3D satranç oyunu. React ve Three.js kullanarak yaptım.

## Ne var içinde?

- 3D ve 2D olmak üzere iki farklı görünüm modu
- Yapay zekaya karşı veya arkadaşınla oynayabilirsin
- Detaylı 3D modeller (kale, at, fil vs.)
- Hamle geçmişi ve alınan taşlar
- Tam satranç kuralları (şah, mat, pat)

## Nasıl çalıştırılır?

```bash
npm install
npm run dev
```

Sonra tarayıcıda `http://localhost:5173` adresine git.

## Oynanış

Taşa tıkla, yeşil karelere hamle yap. Fareyle kamerayı döndürebilirsin, tekerlek ile zoom yapabilirsin.

## Teknolojiler

React, Three.js, Vite kullandım. Masaüstü versiyonu için Electron var.

## Build

Web için:
```bash
npm run build
```

Masaüstü uygulaması için:
```bash
npm run dist
```
